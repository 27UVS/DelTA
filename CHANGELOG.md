## [Unreleased]

### Fixed

- **Доска / окно:** при сильном уменьшении окна приложение могло подвисать или вылетать из‑за накопления потоков загрузки фона (`BackgroundImageLoader`); при выходе — предупреждение «QThread: Destroyed while thread is still running». Исправлено: проверка прерывания в потоке, одна активная загрузка с отложенным повтором, квантование размера, дебаунс ресайза, ожидание потоков при закрытии окна; в `BoardPrefetchThread` учтено прерывание.
- **Профиль (админ):** при сужении окна поля и подписи визуально сжимались по вертикали. Форма обёрнута в прокрутку, убраны лишние вертикальные растяжки, зафиксирована минимальная высота строк ввода и политика размера; в стилях для `QLineEdit` задан `min-height`.

## [v1.1.0] - 2026-04-05

### Added

- **Подзадачи (цепочка):** у задачи можно задать упорядоченный список подпунктов с названием и ответственным; на доске — компактная полоса прогресса (серый/зелёный); при количестве подзадач **> 18** на карточке показывается одна линия доли выполнения без точек.
- **Просмотр задачи:** блок «Цепочка подзадач» с многострочной змейкой, подписями и сглаженными переходами между рядами; ограниченная по высоте зона с прокруткой и кнопка **«Развернуть»** для полного просмотра в отдельном окне.
- **Создание/редактирование:** раздел «Подзадачи» со списком строк (название, ответственный, «Готово», удаление).
- **Настройки интерфейса:** параметр «Подзадач в строке» (для детальной цепочки), сохраняется в `ui_settings.json`.
- **Правила данных:** подзадачи нельзя отмечать выполненными с пропуском (только по порядку с начала цепочки); при сохранении проверяется **startline ≤ deadline**; при **создании** новой задачи в поля подставляются **текущие дата/время** и **+1 сутки** для deadline.
- **Панель «Люди» — фильтр по задачам:** в меню фильтра добавлено подменю **«Задачи»** — порядок списка по числу активных задач (как обычно по ролям, сначала с наибольшим или наименьшим числом задач), отбор **«Все»**, **«Нет задач»** (0 задач) и **«С задачами»**; сочетается с фильтрами по ролям и статусам.
- **Описание задачи:** кнопка снятия ссылки с выделенного текста (убирается гиперссылка, подчёркивание и цвет, как у обычного текста в поле).

### Changed

- **Скроллбары:** плоский современный вид, скруглённый ползунок, настраивается через общую таблицу стилей приложения.
- **Просмотр задачи:** у неактивных полей дат и у ответственных скрыты стрелки раскрытия/степпера; календарь у дат отключён; при нажатии «Редактировать» окно просмотра сразу скрывается, открывается только диалог редактирования.

### Fixed

- Время **Startline** при создании задачи: не сбрасывалось в 00:00 из‑за порядка синхронизации дат и `setSelectedDate` у календаря (установка start/deadline под `blockSignals`, без принудительного `setSelectedDate` при инициализации).
