# DelTA

**DelTA** — настольное приложение для управления и делегирования задач внутри команды: помогает руководителю (и не только) удобно фиксировать задачи, распределять ответственность и отслеживать прогресс.

Проект является **второй версией** приложения **[DelegTa](https://github.com/27UVS/DelegTa)**.

## Возможности

- **Доска задач**: создание задач и ведение их статусов (черновик / в работе / завершено / отложено).
- **Структура и справочники**: роли, статусы, направления/темы (subjects).
- **Профиль и персональные настройки**: базовые настройки пользователя и интерфейса.
- **Локальное хранение данных**: данные сохраняются в папке `db/` рядом с приложением.

## Быстрый старт

### Запуск из исходников

1. Установите Python (рекомендуется 3.12+).
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите приложение:

```bash
python main.py
```

При первом запуске, если папки `db/` нет, она будет создана автоматически из `db_template/`.

### Сборка в exe (Windows)

Скрипт сборки:

```bash
powershell -ExecutionPolicy Bypass -File .\build_exe.ps1
```

Результат появится в:

- `dist\DelTA\DelTA.exe`

## Данные (`db/`)

Папка `db/` содержит рабочие локальные данные (настройки, задачи и т.д.) и **не хранится в репозитории**. Для инициализации используется `db_template/`.
